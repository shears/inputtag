InputTag --Input tags in an input box.

Introduce

It is an jQuery plugin.you can put some tags in an box like input field.you can assign some separative signs to recognize tag.

Document

            
                $(".input-tag").inputTag();
            
        
            
                $(".input-tag").inputTag({
                    separativeSigns: [",", "，", ";", "；",".","。"],
                    multi: true,
                    readonly: false
                });
